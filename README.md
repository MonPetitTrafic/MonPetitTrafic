# 🚄Nani🚌 ?

→📱 **Mon Petit Trafic** est une **application mobile** destinée à l'**information trafic** des lignes de **transport en commun d'Île-de-France**.\
→ 🎯 Le but du projet est de **prévenir l'utilisateur le plus vite possible** lorsqu'une perturbation touche une des lignes de transport qu'il emprunte.

___

### Mais alors, quelle différence avec le service Infos Trafic d'Île-de-France Mobilités (IDFM) ?

Mon Petit Trafic a pour but de répondre à **trois manquements** du service fourni par IDFM :
- 1️⃣ Beaucoup d'annonces, en particulier lorsque qu'elles concernent des légères perturbations (stationnements prolongés, suppression ponctuelle d'une mission, etc) ne sont pas communiquées auprès d'IDFM mais seulement sur X (ex-Twitter).
- 2️⃣ Les annonces sur X sont souvent plus développées que sur la plateforme IDFM.
- 3️⃣ Les opérateurs de lignes de bus se focalisent principalement sur X.

Par ailleurs, un constat fort important :
> Beaucoup de personnes **refusent** (à tort ou à raison) de créer un compte sur la plateforme X, et **perdent donc l'accès** à ces informations précieuses.

Ainsi, vous l'aurez compris, Mon Petit Trafic est développé pour **rendre accessible à tous** les informations trafic diffusées par les opérateurs sur X.

# ⚙️ Fonctionnement

➡️ Mon Petit Trafic se base sur les **flux RSS générés par Nitter**\*, un service open-source permettant d'accéder au contenu de la plateforme X.
- L'utilisateur peut **sélectionner les lignes** dont il souhaite suivre la situation du trafic depuis l'application Mon Petit Trafic.\

*Précision : actuellement, seules les lignes de RER, de Transilien, de Métro et de Tramway sont proposées. Il est prévu d'inclure en plus les lignes de bus ainsi que le Câble C1.*

- Selon les préférences de l'utilisateur, l'application lancera **une notification push-up** dès lors que l'une des lignes sélectionnées est perturbée.
- De même, **chaque évolution de la situation** annoncée sur X sera relayée par notification push-up.
- L'utilisateur peut également **désactiver les notifications**, rendant l'application totalement silencieuse.

\**Noter que ce projet n'est nullement affilié à (l'organisation de) Nitter.*
